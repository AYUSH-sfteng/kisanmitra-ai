# Changelog - KisanMitra AI

All notable changes to the KisanMitra AI Multi-Agent platform will be documented in this file.

---

## [1.0.0] - 2026-06-29
Initial release prepared for the hackathon submission.

### Added
- **Multi-Agent Orchestrator Model**: Parallel querying capabilities and result aggregation.
- **Vite & React Frontend**: Fast, responsive single-page visual layout using Tailwind CSS v4 and Framer Motion.
- **Crop Disease vision classifier**: Complete Early Blight (Alternaria solani) diagnostic panels.
- **Weather, Market & Schemes telemetry**: Hyperlocal wind/precipitation data mapping, 7-day APMC tomato price index graphs, and matching PMFBY subsidies.
- **FastAPI backend mockup**: Full API endpoints matching the frontend payload scheme.
- **Security Auditing Agent**: Real-time prompt injection filtering and PII masking checks.
- **Dashed connection SVG lines**: Flow indicators showcasing active agent threads.
- **Community templates**: README, CONTRIBUTING, LICENSE, CODE_OF_CONDUCT, SECURITY.
- **GitHub Action workflows**: Build validation and lint checking.
