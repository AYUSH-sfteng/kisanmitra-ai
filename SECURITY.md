# Security Policy - KisanMitra AI

We take the security and safety alignment of our Multi-Agent network seriously. If you discover a prompt injection flaw, PII leak vector, or API routing bug, please follow these guidelines to report them.

---

## 1. Supported Versions

| Version | Supported |
| :--- | :--- |
| 1.0.x | :white_check_mark: Yes |
| < 1.0.0 | :x: No |

---

## 2. Reporting a Vulnerability

1. Do NOT open a public GitHub issue.
2. Email your detailed security report to `security@kisanmitra-ai.org` (simulated).
3. Include the following details in your report:
   - Description of the injection payload or bug.
   - Screenshots, terminal commands, or response logs.
   - Steps to reproduce the issue.
4. We will review the bug and release a hotfix within 48 hours.
