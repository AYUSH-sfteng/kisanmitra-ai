# KisanMitra AI Frontend Client

React client interface for KisanMitra AI. 

## Tech Stack
- **React (v19)**: Core UI library.
- **Tailwind CSS v4**: Theme styles and layout.
- **Framer Motion**: Parallel agent transitions and page scrolls.
- **Recharts**: tomato crop APMC price trends index.
- **Lucide React**: Vector inline iconography.

---

## Getting Started

### Prerequisites
- Node.js 18 or higher

### Installation
From the root of the project:
```bash
cd frontend
npm install --legacy-peer-deps
```

### Development
Start the local development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build
Compile the client bundle for production:
```bash
npm run build
```
The output assets will be generated in `frontend/dist/`.
